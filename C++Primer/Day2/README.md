**0x01 无符号与有符号运算 **

​	如果无符号类型的数值与有符号类型的数值进行运算，那么会出现怎么样的类型转换？

```c++
unsigned a = 10;
int b = -42;
cout << a+b << endl;
//output: 4294967264

unsigned a = 98;
int b = -72;
cout << a+b << endl;
//output 26 
```

经过上面两个程序段的结果与我们预计的不太一样，分析：

​	首先将有符号整型转变成无符号然后再做加运算。所以第一个例子会出先一个十分大的数字（$2^{32}-42+10$）。

​	第二个例子，由于 `98 - 72 > 0` 所以溢出，准确来说被截断了，原有的首位1，被截断了。
